# AwakeMac v1.0.0 / 醒着 v1.0.0

首个公开正式版。AwakeMac 是一款原生 macOS 菜单栏工具，可阻止自动息屏和系统空闲睡眠，并在关闭后恢复原有系统设置。

主要功能：

- 无限期或按时长保持唤醒
- “离开一会”：保持唤醒并临时降低亮度，结束后自动恢复
- 实验性的合盖运行模式，带 90 秒安全租约、电量和温度保护
- MacBook/桌面 Mac 能力识别
- 桌面小组件
- 简体中文和英文界面
- Universal 2，支持 Apple Silicon 与 Intel，要求 macOS 14 或更高版本

安装：下载 DMG，将 AwakeMac 拖入 Applications。由于当前公开构建尚未完成 Apple Developer ID 公证，首次启动请右键 AwakeMac →“打开”→ 再次确认。

---

First public stable release. AwakeMac is a native macOS menu bar utility that prevents idle display and system sleep without changing your original Energy settings.

Download the DMG and drag AwakeMac to Applications. This open-source build is not yet notarized with an Apple Developer ID; on first launch, Control-click AwakeMac, choose Open, and confirm once.
